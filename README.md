# ORM-E-commerce-Back-End

## Description

This purpose of this project is to build the back end for a e-commerce site that works with an Express.js API configured to use Sequelize to interact with a MySQL database.

## Installation

- run `source schema.sql`
- run `node seeds index.js`
- run `npm install`
- run `npm start`

## Usage

Use `insomnia` or `postman` execute the api calls to achieve the following:

PRODUCTS:
- Get all Products from DB
- Get Product by ID
- Create a Product
- Update a Product
- Delete a Prodcut

CATEGORY:
- Get all Categories
- Get Category by ID
- Create a Category
- Update a Category
- Delete a Category

TAGS:
- Get all Tags
- Get Tag by ID
- Create a Tag
- Update a Tag
- Delete a Tag
